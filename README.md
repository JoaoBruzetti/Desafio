# Desafio

1. Rodar comando para subir containers

    **`docker compose up -d`**

2. Entrar na pasta backend

    **`cd /backend`**

3. Gerar .env

    **`cp .env.example .env`**

4. Composer install

    **`composer install`**

5. Gerar chave 

    **`php artisan key:generate`**

5. Inicial laravel

    **`php artisan serve`**

6. Rodar migrations

    **`php artisan migrate`**
